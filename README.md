# 🚀 CI/CD Status Dashboard

A real-time status dashboard with automated CI/CD 
pipeline using GitHub Actions and Docker.

## 🔗 Live Demo
- Frontend: https://cicd-dashboard-xxxx.vercel.app
- Backend: https://cicd-dashboard-backend-phsn.onrender.com

## 🛠️ Tech Stack
- React.js (Frontend)
- Node.js + Express (Backend)
- Docker (Containerization)
- GitHub Actions (CI/CD Pipeline)
- Vercel (Frontend Hosting)
- Render (Backend Hosting)

## ✅ Features
- Live service status monitoring
- Auto refresh every 30 seconds
- Server information display
- Automated CI/CD pipeline
- Docker containerized app
- Zero manual deployment

## 🔄 How CI/CD Works
1. Push code to GitHub
2. GitHub Actions triggers automatically
3. Tests run automatically
4. Docker images build automatically
5. App deploys automatically

## 🚀 How to Run Locally
- Clone the repo
- cd backend → npm install → node server.js
- cd frontend → npm install → npm start

## 📸 Pipeline Flow
Push Code → Tests → Docker Build → Deploy ✅
